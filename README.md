# urlShortener

## Install the dependencies

- npm install

## Start the server

- npm start

## Requests

- Create a new short url

```
Type: Post
localhost:3333/url
body: {"urlO": "site"}
```

- Get Url list

```
Type: Get
localhost:3333/
```

- Be redirect

```
Type: Get
localhost:3333/"short"
```
